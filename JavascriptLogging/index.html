Welcome to Logging 101

This project explores the various methods attached to the console object.

Console functions NOT used: group, groupCollapsed, trace

Common

  • console.log
  • console.info
  • console.warn
  • console.error

Less common

  • console.time / console.timeend
  • console.assert
  • console.clear
  • console.count
  • console.table
